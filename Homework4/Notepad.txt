5)	what is head, body, html element and why do we need them in the page?
Head , body 

The HTML <head> element provides general information (metadata) about the document, including its title and links to its scripts and style sheets.
<html>
<head>
<title>Document title</title>
</head>
</html>

The HTML <body> Element represents the content of an HTML document. There can be only one <body> element in a document.

<html>
<head>
<title>Document title</title>
</head>
<body>
<p>This is a paragraph</p>
</body>
</html>

An HTML element usually consists of a start tag and end tag, with the content inserted in between and it can be said to be everything from the start tag to end tag. Examples of elements are
<head>, <title>, <body>, <header>, <article>, <section>, <p>, <div>, <span>, <img>, and so on.

So, in order to have a meaningful content oriented web page, these play a pivotal role to get the end result.
